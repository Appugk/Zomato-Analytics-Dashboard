# 📊 Zomato Data Analytics Project  
### Restaurant Performance & Market Analysis  

## 📌 Overview  
This project analyzes Zomato restaurant data to uncover insights into customer preferences, restaurant performance, and market trends. The goal is to support data-driven decision-making using interactive dashboards and visualizations.

## 🛠️ Tools & Technologies  
- Excel  
- Power BI  
- Tableau  
- MySQL  

## 📂 Project Workflow  
1. Data Collection  
2. Data Cleaning & Preprocessing  
3. Data Analysis using SQL  
4. Data Visualization (Power BI & Tableau)  
5. Dashboard Creation

## 📊 Key Insights  
- Analysis of restaurant ratings and customer votes  
- Delivery and table booking trends  
- City-wise and country-wise performance  
- Identification of high-performing restaurants  
- Customer behavior and preferences  

## 📈 Features  
- Interactive dashboards  
- Visual representation of key metrics  
- Easy-to-understand insights for business decisions  

## 🎯 Objective  
To analyze food delivery and restaurant data to improve customer experience, optimize restaurant performance, and identify market opportunities.
